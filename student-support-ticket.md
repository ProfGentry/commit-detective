# Student Support Ticket

## Ticket ID

ST-1001

## Student Issue

The student reports that they cannot access their GitHub repository from Git Bash.

## Reported Error

Permission denied.

## Initial Questions

- Has the student created a GitHub account?
- Is Git installed on the computer?
- Is the student using HTTPS or SSH?

## Suggested First Step

Verify the remote repository URL by running:

```bash
git remote -v
```
