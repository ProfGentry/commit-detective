# Troubleshooting Notes

## Login Issues

If a student cannot log in:

1. Confirm the username is correct.
2. Check that Caps Lock is not enabled.
3. Reset the password if needed.
4. Try logging in from another browser.

## Network Issues

If the internet is not working:

1. Check the network cable or Wi-Fi connection.
2. Restart the device.
3. Confirm the device has an IP address.
4. Contact support if the issue continues.

## Software Issues

If an application will not open:

1. Restart the computer.
2. Check for updates.
3. Reinstall the application if necessary.
