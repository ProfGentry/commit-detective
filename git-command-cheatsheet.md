# Git Command Cheatsheet

## Check Repository Status

```bash
git status
```

Shows which files have changed and whether they are staged.

## Stage a File

```bash
git add filename.md
```

Adds a file to the staging area.

## Commit Changes

```bash
git commit -m "Add clear message here"
```

Saves a snapshot of the staged changes.

## Check Git Information

```bash
git stats
```

Shows the current repository status.
